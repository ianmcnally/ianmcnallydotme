---
title: "Get that div all the way down"
date: "2014-05-20"
---

Say your want a div (i.e., .container) to extend all the way down the page, until about 20px before the bottom. Check it out:

```
.container {
    position: absolute;
    top: $your_top_value;
    bottom: 20px;
}

```

Demo [here](http://codepen.io/imcnally/pen/ieCDp).
