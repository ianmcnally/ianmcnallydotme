Today, I used box-shadow to create an overlay!

Notice the lookahead list (bags, baby bags). It's a `<ul>`.

On that `<ul>` I put a box-shadow:

```css
ul {
  box-shadow: 0 0 0 100vh rgba(0, 0, 0, 0.4);
}
```

And it worked famously!
